Clinical Instruction API

End Points

There is only one endpoint available for this API and this can be accessed via the, /instruct endpoint. The endpoint takes the parameter prompt which instructs the fine-tuned 355M GPT2 LLM the instruction to follow. Further explanation is given below.

/instruct POST

Parameter Type Position # Description
prompt string body Required Combined nurse profile and clinical scenario, including the nurse's introduction and the clinical situation with a specific question.
nursing_competency string body Optional Area of nursing competency relevant to the scenario
max_num_tokens int body Optional The maximum number of tokens to be generated by the model. Defaults to 100. Recommended max_num_tokens=1000 if GPU is enabled
temperature int body Optional Controls the randomness and creativity of the model's output.
Temperature default(1. Recommended)
top_k int body Optional Limits the model's choices when generating text to the k most likely next tokens.
Default: 5.

Response

{
    "status": true,
    "result_code": 200,
    "message": "Success!",
    "values": {
        "Response": "model generated text",
    }
}
